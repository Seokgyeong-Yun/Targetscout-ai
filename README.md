# TargetScout AI

Bio-AI platform for antibody target evaluation and research intelligence.

## Overview

TargetScout AI is a web application designed for researchers working in antibody engineering, oncology, and drug discovery.

The platform helps users evaluate a therapeutic target by integrating information from multiple public biomedical databases.

Users can search a target gene or protein (e.g., MSLN, HER3, TROP2, CLDN18.2) and quickly explore:

* Target overview
* Disease associations
* Literature landscape
* Clinical trial status
* Competitive antibody landscape
* Patent resources
* Research trend monitoring

The goal is to reduce the time required for early-stage target assessment and provide a unified research dashboard for target discovery.

---

## Motivation

As a graduate student in antibody engineering and drug discovery, I frequently spend significant time collecting information from multiple databases before evaluating a therapeutic target.

TargetScout AI was developed to streamline this process by bringing key information into a single platform and generating a concise target assessment summary.

---

## Data Sources

This project utilizes publicly available biomedical resources:

* PubMed
* ClinicalTrials.gov
* UniProt
* NCBI Gene
* Human Protein Atlas (HPA)
* Open Targets Platform
* Google Patents (search links)

All data remain the property of their respective providers.

## Features (MVP v1)

### Target Overview

Basic information about the target gene and protein.

### Literature Explorer

Recent publications related to the target.

### Clinical Landscape

Relevant clinical trials and development status.

### Competitive Landscape

Known antibodies and therapeutic programs targeting the selected antigen.

### Target Suitability Summary

AI-assisted summary highlighting opportunities and risks.
